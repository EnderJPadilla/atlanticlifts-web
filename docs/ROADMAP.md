# Atlanticlifts - ROADMAP.md

## Objetivo

Este documento define el orden oficial de implementación del proyecto.

Claude Code deberá seguir estrictamente este roadmap.

No deberá avanzar a una fase posterior hasta completar y validar completamente la fase actual.

Cada tarea deberá marcarse como completada únicamente cuando:

- Compile sin errores.
- Pase ESLint.
- Pase TypeScript.
- Sea responsive.
- Funcione en Light y Dark Mode.
- Cumpla accesibilidad.
- Mantenga consistencia con PROJECT.md.

# Regla Final

La calidad tiene prioridad sobre la velocidad.

Es preferible completar menos tareas con excelencia que avanzar rápidamente generando deuda técnica.

Cada fase deberá dejar el proyecto en un estado estable, funcional y listo para continuar con la siguiente.
